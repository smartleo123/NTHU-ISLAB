# 三大法人選擇權未平倉多空差(空)

## 計算方式
- 多空差 = sum(多方未平倉餘額口數) - sum(空方未平倉餘額口數)
  - 多方為買call+賣put，空方為買put+賣call
  - rolling window求百分比

## 日K(多)_1224_p15

進場條件：
- 多空差百分比小於39(rolling days 30日)且多空差為連續11日最低點



出場條件：
- 多空差百分比大於53(rolling days 110日)或多空差為連續16日最高點
- 固定停損FIXLOSS = 75
- HL停利(20,30)


樣本時間：
- 樣本內：2009/1/1 - 2017/12/31
- 樣本外：2018/1/1 - 2020/12/24
