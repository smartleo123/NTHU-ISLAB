# 三大法人選擇權未平倉多空差

## 計算方式
- 多空差 = sum(多方未平倉餘額契約金額) - sum(空方未平倉餘額契約金額)
  - 多方為買call，空方為買put
  - rolling window求百分比

## 日K(多)_0226_p11

進場條件：
- 多空差百分比大於54（rolloing days 110日）


出場條件：
- 多空差為連續60日最低點
- ATR停損（len：25 日, productvalue：1.75）
- HL停利（15, 25）


樣本時間：
- 樣本內：2009/1/1 - 2017/12/31
- 樣本外：2018/1/1 - 2021/02/25
